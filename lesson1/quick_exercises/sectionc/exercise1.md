### a. Using the REPL, create a string variable using single quotes.

```python
In [4]: my_string = 'hello world'

In [5]: print(my_string)
hello world
```

### b. Using the REPL, create a string variable using double quotes.

```python
In [6]: my_string2 = "something else"

In [7]: print(my_string2)

```
