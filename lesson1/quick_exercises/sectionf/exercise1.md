
### a. Create a list named `my_list` that contains four different strings.

```python
In [1]: my_list = ["This", "is", "a", "list"]
```

```python
In [2]: my_list[0]
Out[2]: 'This'

In [3]: my_list[2]
Out[3]: 'a'

In [4]: my_list[-1]
Out[4]: 'list'
```
